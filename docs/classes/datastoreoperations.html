Class DatastoreOperations<T>

This class contains some lower-level abstractions for the LevelDB.

Usage

You should not instantiate this class, consider using Datastore instead. After that, you can access this class through datastore.methods.

createReadStream

  • createReadStream<I>(onData: function): Promise<I[]>
    • Defined in database/datastore/datastore.methods.ts:122

    Created a read stream on LevelDB database. onData callback will be called on each object, and if the callback returns non-null value, it will add this value to an array and return it later, when the stream will end.

    Usage

    const allObjects = await this.createReadStream<T>(data => data.value)

    Type parameters

    • I

      Type of the values that are added to an array.

    Parameters

    • onData: function

      Your own callback. If the callback returns value of type I or non-null value, it will add the returned value to an array.
